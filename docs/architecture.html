Azure Event Hubs Streaming Platform Module

Azure Event Streaming

Executive Summary

Terraform module for deploying Azure Event Hubs, a fully managed real-time event streaming platform. Provisions an Event Hub namespace with configurable SKU (Basic/Standard/Premium), multiple event hubs with partition counts and message retention, consumer groups for parallel processing, namespace-level authorization rules with granular listen/send/manage permissions, event capture to Azure Blob Storage (Avro format), auto-inflate for automatic throughput scaling, zone redundancy, and optional private endpoints with DNS zone integration.

Data Flow

1
Producers send events to specific event hubs within the namespace using SAS keys or AAD.
2
Events are distributed across partitions for parallel processing and ordering guarantees.
3
Consumer groups allow multiple consumers to independently read from the same event hub.
4
Each consumer maintains its own offset checkpoint for exactly-once processing.
5
Event Capture automatically archives events to Azure Blob Storage in Avro format.
6
Auto-inflate scales throughput units dynamically based on ingestion load.

Production Readiness Checklist

  • Choose appropriate SKU tier (Standard or Premium)
  • Enable auto-inflate for variable workloads
  • Set partition count based on consumer parallelism
  • Configure message retention for replay requirements
  • Enable event capture for audit/compliance
  • Enable zone redundancy for high availability
  • Create dedicated consumer groups per consumer application
  • Configure authorization rules with least privilege
  • Enable private endpoint for network isolation
  • Monitor throughput units and consumer lag

Deployment

terraform init
terraform plan -out=tfplan
terraform apply tfplan

# Verify namespace
az eventhubs namespace show \
  --name my-eventhub-ns \
  --resource-group rg-eventhub \
  --query "{sku:sku.name,status:status}" -o table

# List event hubs
az eventhubs eventhub list \
  --namespace-name my-eventhub-ns \
  --resource-group rg-eventhub -o table
